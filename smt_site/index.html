Troubleshoot
Not printing? Try these in order:
1. Type USB001 in the manual box above → click Test Print
2. Type USB002 if USB001 fails
3. Check the black server window — it shows detailed errors
4. Open localhost:44444/debug to verify server is working
⚡
Print server not running.
Start it once to enable fully automatic silent printing. Without it, the browser print dialog will appear for each tag.
How to start the print server (one-time setup): 1. Make sure Node.js is installed on this PC ↗ nodejs.org
2. Copy printer-server.js to any folder
3. Open Command Prompt in that folder and run:
node printer-server.js 4. Leave that window open — then come back and click Retry below
